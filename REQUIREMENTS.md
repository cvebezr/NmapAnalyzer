# Installation Guide for Nmap Scan Analyzer

## Prerequisites

### 1. Python Requirements
- Python 3.6 or higher
- pip (Python package manager)

### 2. System Requirements
- Nmap scanner
- xsltproc (for HTML report generation)

## Installation Steps

### Step 1: Install System Dependencies

#### Ubuntu/Debian:
```bash
sudo apt update
sudo apt install -y nmap xsltproc python3 python3-pip
